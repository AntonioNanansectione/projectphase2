Tasty Bites Restaurant Website

Overview
The Tasty Bites Restaurant Website is a responsive and interactive web application designed using HTML, CSS, JavaScript, jQuery, and Bootstrap. It showcases various features such as a navigation bar, menu section with cards, smooth scrolling to sections, and a contact form with AJAX form submission.

File Structure
index.html: Home page of the website.
menu.html: Menu page displaying food items using Bootstrap cards.
gallery.html: Gallery page showcasing restaurant images.
contact.html: Contact page with a form for user inquiries.
styles.css: Custom CSS file for styling the website.
scripts.js: JavaScript/jQuery file for interactive elements and form submission.
burger.jpg: Image file used in the menu section.
gallery1.jpg, gallery2.jpg: Image files used in the gallery section.

Usage
Navigation: Navigate through different pages using the navbar links.
Menu: Explore the delicious dishes available at Tasty Bites.
Gallery: View stunning images of the restaurant and its ambiance.
Contact: Submit inquiries or feedback using the contact form.
